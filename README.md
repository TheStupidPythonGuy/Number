# Number
A number guessing game

## Features

- Simple
- Fun to play
- Easy, Medium, Hard, or Impossible

## Requirements

- Python 3.x

## Installation

Run in Terminal using **chmod +x /path/to/Number Guessing Game.py**
Then, use **./path/to/Number Guessing Game**
